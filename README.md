# Work-Day-Planner

## Purpose
A website that allows the user to make a daily schedule of tasks for each hour of the work day.
As the hours progress throughout the day, the colors change to reflect past, present and future condition.

## Built With
* HTML
* CSS
* JavaScript
* JQuery
* Bootstrap
